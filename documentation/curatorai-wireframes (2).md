## Color Usage Guide

### Primary Actions
- Buttons: `#6366F1` (Indigo)
- Hover: `#4F46E5` (Darker Indigo)
- Active states and selected items

### Secondary Actions
- Buttons: `#EC4899` (Pink)
- Accent elements
- Special features (camera, upload)

### Status Indicators
- Success: `#10B981` (Green) - Completed, Available
- Warning: `#F59E0B` (Amber) - Pending, Review
- Error: `#EF4444` (Red) - Delete, Ban, High Priority
- Info: `#3B82F6` (Blue) - Notifications, Info

### Backgrounds
- Main: `#F9FAFB` (Light Gray)
- Cards: `#FFFFFF` (White)
- Hover: `#F3F4F6` (Slightly darker gray)

### Text
- Primary: `#111827` (Dark Gray)
- Secondary: `#6B7280` (Medium Gray)
- Disabled: `#D1D5DB` (Light Gray)

### Borders
- Default: `#E5E7EB` (Light Border)
- Focus: `#6366F1` (Indigo)
- Error: `#EF4444` (Red)
